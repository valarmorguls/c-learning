#为什么虚函数可以跟着对象走？
对象占用存储区，头八个字节不一样，头八个字节指向虚函数表的地方（vtable），虚函数表装着
每一个虚函数。
拥有虚函数的对象和没有拥有虚函数对象的大小不一样

#每一个类有一个虚函数表
相同类型的对象指向 相同的虚函数表
虚函数表是存储函数地址的数组
